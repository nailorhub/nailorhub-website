# NailorHub Website

Private repository for the NailorHub business website. Built with React + Vite + Tailwind CSS. For internal development only.

Run locally:
- `npm install`
- `npm run dev`

Build for production:
- `npm run build`
- `npm run preview`

Project layout: `src/` (site code), `public/` (static assets), `index.html` (entry point).
